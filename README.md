# Gym App 🏋️‍♂️

A desktop application built with **Electron.js** for gym subscription and management, featuring **ReactJS** frontend, **MongoDB** database, and support for **Arabic translation**.

## 📌 Features
- Cross-platform **desktop app** (Windows, macOS, Linux via Electron.js)
- **ReactJS** frontend with **React Router** for navigation
- **MongoDB** database for storing gym member data
- **IPC (Inter-Process Communication)** for backend integration
- **Axios** for fetching and sending data
- **TailwindCSS** for responsive and modern styling
- **i18n** for English/Arabic language support
- **MUI Icons** for a polished interface

## 🛠️ Tech Stack
- **Frontend:** ReactJS, React Router, TailwindCSS, MUI Icons
- **Backend:** Electron.js, IPC, MongoDB
- **Other Tools:** Axios, i18n, Vite

## 🚀 Installation

### Prerequisites
- [Node.js](https://nodejs.org/) installed
- [MongoDB](https://www.mongodb.com/) running locally or remotely

### Steps
1. **Clone the repository**
   ```bash
   git clone https://github.com/abderrahm4ne/gym-app.git
   cd gym-app
2. Install dependencies
    ```bash
    npm i
3. Run the app in development mode
    ```bash
    npm run start
4. Build the production version
   ```bash
    npm run build
